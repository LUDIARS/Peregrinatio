# 拠点名と設備チェック

## 目的

- 拠点はしおり上で読みやすい8文字以内の名前を使う。
- 正式な場所名は変更せず、場所詳細から旅行ごとの拠点名を編集できる。
- 拠点や複合施設に含まれる設備・施設を列挙し、旅行でやりたいものを選べる。

## 動作

拠点を設定すると正式名から8文字以内の初期名を即時設定し、その後 Haiku が短い拠点名と設備候補を提案する。Haiku が利用できない場合も拠点設定自体は完了し、画面に自動提案の失敗を表示する。

場所詳細では拠点名を8文字以内で編集できる。設備候補は場所単位で共有し、チェック状態は旅行ごとに保存する。複合施設は Google Places のカテゴリまたは名称から判定し、詳細表示時に候補が空なら一度だけ自動提案する。
