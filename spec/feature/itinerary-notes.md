# 旅のしおりメモ

## 目的

旅の予定を確認している画面から離れずに、その日全体の注意事項と、訪問場所ごとの情報を確認・編集できるようにする。

## 振る舞い

- 日別メモは `trip_days.notes` に保存する。
- 場所メモは共有場所の `places.notes` に保存し、同じ場所が複数の日程に登場する場合はすべて同じ内容を表示する。
- 保存時は前後の空白を除去し、空になったメモは `null` として保存する。
- 保存に失敗した場合は編集内容を保持し、各編集欄の近くにエラーを表示する。

## レスポンシブ表示

- PC (`min-width: 900px`): 各日の予定列の右側に、その日の日別メモを表示する。
- スマホ (`max-width: 899px`): 場所メモは各場所カード内に表示し、日別メモは各日の予定の後に表示する。
- スマホでは左下の「日別メモ」ボタンから、全日分をまとめたオーバーレイも開ける。
